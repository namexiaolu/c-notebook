sizeof 返回变量的类型所占的大小。 int 就是4 char就是1


``` c++
cout <<sizeof(char) << endl; _1__ 

cout << sizeof(short) << endl; __2_ 

cout << sizeof(int) << endl; __4_ 

cout << sizeof(float) << endl; _4__ 

cout << sizeof(double) << endl; __8_ 
```
